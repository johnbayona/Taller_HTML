En este proyecto desarrollé una landing page utilizando Flexbox para organizar de forma sencilla la barra de navegación, la sección principal y las tarjetas de contenido. Para que la página se adaptara mejor a diferentes tamaños de pantalla, utilicé varias unidades de medida como **px, %, rem, em, vw y vh**. Además, apliqué **flex-wrap** para que las tarjetas se acomodaran automáticamente cuando no hubiera suficiente espacio. También hice uso de **justify-content**, **align-items** y **gap** para distribuir los elementos de manera ordenada. El botón de WhatsApp se mantuvo siempre visible gracias a **position: fixed**, mientras que la etiqueta "Nuevo" sobre la imagen se logró combinando **position: relative** y **position: absolute**. Finalmente, utilicé la propiedad **order** para resaltar una de las tarjetas y darle mayor importancia dentro de la galería.
