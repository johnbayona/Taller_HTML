Laboratorio del Modelo de Cajas en CSS

Content

Esta caja muestra únicamente el contenido con un ancho fijo. Sirve para observar el espacio que ocupa el contenido antes de aplicar otras propiedades del modelo de cajas.

Padding

El padding agrega espacio interno entre el contenido y el borde, mejorando la presentación y la legibilidad del elemento.

Border

El borde delimita visualmente la caja y permite diferenciar claramente cada componente dentro de la página.

Margin

El margin crea separación externa entre esta caja y los demás elementos del documento HTML.

Border Radius

La propiedad border-radius redondea las esquinas del borde, proporcionando un diseño más moderno y agradable.

Overflow: Visible

Este texto es demasiado largo para el tamaño del contenedor. El contenido continúa mostrándose fuera de la caja porque el valor visible permite que el contenido desborde el área establecida sin ocultarlo.

Overflow: Hidden

Este contenido supera la altura del contenedor, pero la parte sobrante queda oculta al utilizar la propiedad overflow con el valor hidden.

Overflow: Scroll

Cuando el contenido es mayor que el tamaño del contenedor aparecen barras de desplazamiento, permitiendo visualizar toda la información mediante el scroll.

Overflow: Auto

El navegador mostrará barras de desplazamiento únicamente cuando sean necesarias, manteniendo un diseño más limpio.
