Bootcamp de Desarrollo Web Frontend

Información General del Curso

Este módulo está orientado al aprendizaje de HTML5 y CSS3, fortaleciendo las habilidades para construir interfaces web modernas mediante buenas prácticas de diseño y organización.

Duración del módulo: 20 horas de formación práctica.

Cronograma del Módulo

Sesión Tema Duración Instructor
1 Introducción a HTML5 2 horas Ana Martínez
2 Etiquetas Semánticas 3 horas Carlos Gómez
3 CSS Básico 3 horas Laura Pérez
4 Modelo de Cajas 3 horas Juan Rodríguez
5 Proyecto Integrador 4 horas María Sánchez

Recomendaciones

  • Practicar diariamente con HTML y CSS.
  • Utilizar nombres descriptivos para clases y archivos.
  • Aplicar correctamente el modelo de cajas.
  • Comentar el código cuando sea necesario.
  • Validar el código antes de entregar el proyecto.

Ejemplo del Modelo de Cajas

Este contenedor tiene un texto bastante extenso para demostrar el funcionamiento de la propiedad overflow. Cuando el contenido supera la altura definida, aparecerá una barra de desplazamiento automática que permitirá visualizar toda la información sin afectar el diseño de la página. Este comportamiento es muy útil cuando se presentan descripciones largas, comentarios o información adicional dentro de un espacio limitado.
