Curso de Desarrollo Web Frontend

Clase 5 - Tipografía, Tablas y Modelo de Cajas

En esta sesión los estudiantes aprenderán a mejorar la presentación visual de una página web mediante el uso de propiedades tipográficas, tablas estilizadas y el modelo de cajas de CSS.

Objetivo: crear interfaces más organizadas, legibles y profesionales.
